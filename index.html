Usage

1. Include the listy-text.css and listy-text.js files on your page (duh!):

<script src="listy-text.js" type="text/javascript"></script>
<link rel="stylesheet" type="text/css" href="listy-text.css">

2. Add the Listy-Text input html:

<ul class="listy-text">
<li class="listy-text-box">
  <input class="text-value" type="text" />
  <input name="csv-format" type="hidden" />
</li>
</ul>
  • To specify a width for the input, add a size class such as input-large (bootstrap) to the parent ul element.
  • To use one of the included themes above, include their css files and add either the bootstrappy or facebooky class to the parent ul element.

3. Grab the listy-data in one of two available formats:

  • As a single hidden form input with all items (comma seperated) as the value.
    <input name="csv-format" type="hidden" value=",a,b,c">
  • As multiple hidden form inputs for each item entered.
    <input type="hidden" class="individual-format" name="list-item1" value="a">
    <input type="hidden" class="individual-format" name="list-item1" value="b">
    <input type="hidden" class="individual-format" name="list-item1" value="c">
